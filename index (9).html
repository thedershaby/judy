Project 01 · Database Design

ERD PROJECT — RESTAURANT SYSTEM

This project focused on translating real restaurant workflows into a structured database design. The system models how restaurants manage customers, orders, menu items, reservations, and payments while ensuring data consistency and scalability.

Tasks Performed

  • Conducted requirements analysis based on restaurant operations
  • Normalized tables to reduce redundancy and improve efficiency
  • Created a detailed ERD diagram using proper database conventions
  • Implemented the database structure in Microsoft Access
  • Tested the database using sample restaurant transactions

Skills Demonstrated

  • ERD design (entities, relationships, keys)
  • Database implementation in Microsoft Access
  • Normalization and data organization
  • Query creation and basic forms/reports

Connection to Goals

Applies database design concepts in practice and strengthens skills in building and managing relational databases using Microsoft Access — a core foundation for a career in Business Informatics.
